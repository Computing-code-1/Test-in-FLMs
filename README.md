# Identification-Robust Testing in Endogenous Functional Linear Regression

This repository contains the R code used for the simulation and empirical results in the paper:

**Identification-Robust Testing in Endogenous Functional Linear Regression with Weak or Irrelevant Auxiliary Variables**

## Files

The main R scripts are:

* `Simulation_version6_unif_based_nointercept...R`
  Simulation code for the model without an intercept.

* `Simulation_version6_unif_based_withintercept...R`
  Simulation code for the model with an intercept.

* `Simulation_comparison_ver4_modified...R`
  Simulation code for the comparison with the functional IV test of Seong and Seo (2025).

* `empirical_analysis_pvalue_table.R`
  Empirical analysis code for the electricity demand and temperature distribution application.

The following auxiliary R functions are required:

* `inprod.R`
* `operator.R`
* `bandfn.R`
* `lr_var_v2.R`
* `covtoeigennew2.R`

## Data availability

The empirical analysis uses the following MATLAB data files:

* `Elec_demand_data.mat`
* `KTemp_Density.mat`
* `Temp_Quant_v202501.mat`

These data files were generated by Kyungsik Nam for Nam and Seo (2025, *Energy Economics*) and were used in this paper with his permission. They are not included in this public repository because redistribution permission has not been granted. Kyungsik Nam has generously agreed to provide the data for review and replication purposes, so please request access directly from him.

To run the empirical analysis, place the three `.mat` files in the working directory specified in `empirical_analysis_pvalue_table.R`.

The simulation scripts do not require these empirical data files.

## Reproducing the results

Before running the scripts, update the working directories in the code to match the local folder structure.

### Simulations

Run the relevant simulation scripts directly in R. The scripts report the rejection rates stored in memory. Local saving commands are commented out in the public version of the code.

For the main simulation tables, the parameter `upermean` corresponds to (\beta_u) in the paper, and `kernf` determines the kernel:

* `kernf = 7`: Bartlett kernel
* `kernf = 1`: Parzen kernel

For the comparison table with the existing functional IV test, the variable `goodIV` determines the design:

* `goodIV = 1`: informative design
* `goodIV = 0`: weakly informative design

### Empirical analysis

Run `empirical_analysis_pvalue_table.R` after placing the required `.mat` files in the working directory. The script computes the empirical p-values and produces the temperature density/quantile function plot in the R plotting window.

## Notes

The code is written to reproduce the numerical results reported in the paper. Some local file-saving commands used during the preparation of the manuscript have been commented out in the public version.
